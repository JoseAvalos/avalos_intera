Configure Ubuntu

1.-Install ROS

Official information could be find:

  • http://sdk.rethinkrobotics.com/intera/Main_Page
  • http://sdk.rethinkrobotics.com/intera/Workstation_Setup

We set the step in order to install ROS.

Setup your sources.list

sudo sh -c 'echo "deb http://packages.ros.org/ros/ubuntu xenial main" > /etc/apt/sources.list.d/ros-latest.list'

Setup your keys

sudo apt-key adv --keyserver hkp://ha.pool.sks-keyservers.net:80 --recv-key 421C365BD9FF1F717815A3895523BAEEB01FA116

Update to Latest Software Lists

sudo apt-get update

Install ROS Kinetic Desktop Full

sudo apt-get install ros-kinetic-desktop-full

2.- Create Workspace

Directory name will depende of the user. Directory could change according to your preferences in my case will be avalos_ros_ws.

mkdir -p ~/avalos_ros_ws/src
source /opt/ros/kinetic/setup.bash
cd ~/avalos_ros_ws
catkin_make

3.-Install INTERA Robot SDK

This git will contain the default software to use the robot.

cd ~/avalos_ros_ws/src
wstool init .
git clone https://github.com/RethinkRobotics/sawyer_robot.git
wstool merge sawyer_robot/sawyer_robot.rosinstall
wstool update

Build again

cd ~/avalos_ros_ws
catkin_make

4.-Add to Bash

In order to use roscd you have to add your directory to your bash file. Using Sublime :

subl ~/.bashrc 
-- add at the end
source /opt/ros/kinetic/setup.bash
source /home/jose/avalos_ros_ws/devel/setup.bash

To check you could try echo $ROS_PACKAGE_PATH in terminal.

5.- Define envioroment

We have to copy a default file wich contain the network connection information.

cp ~/avalos_ros_ws/src/intera_sdk/intera.sh ~/avalos_ros_ws

To edit the file I reccommed ~Sublime~ https://www.sublimetext.com/3

Change version in *.sh file

A important point is define the ROS version. In this work, it is kinectic

ros_version="kinetic"

In order to move fast to this directories, we define :

test=/home/jose/avalos_ros_ws/src/intera_sdk/avalos_intera/scripts/test
avalos=/home/jose/avalos_ros_ws/src/intera_sdk/avalos_intera

AVADHI Conecction

This is used when the robot don't generate a IP. Refresh withouth close all envioroments. This is nedded to add for python msg

source ~/avalos_ros_ws/devel/setup.bash

In order to conect. We have to disable the internet

sudo avahi-autoipd  enp0s31f6

In another window open:

avahi-browse -a -r

1.-Enter to boot interface

First of all, you need a keyboard to access to boot actions.
You have to connect it through the USB y keep press Ctrl+f after press boton ON. When you see a regressive count (like 5,4,...) you have to keep press Crtl and press and depress f. You have to wait until a white scree white red "Preparing to boot... , in this momento you have to press Ctrl+f to access to the menu.
