Welcome

Hello! My name is Jose and the goal of this website is follow my thesis development. Also, It will work as a fast guide to introduce and help student with the Robot Sawyer and ROS envioroment. This work is taking in my semesters 2018 I - II. Where my personal computer-workstation has the following characteristics:

COMPONENT DESCRIPTION
S.O. Ubuntu 16.04
ROS Kinetic
Processor Intel® Core™ i7-7500U CPU @ 2.70GHz × 4
Dedicated Solid State 100Gb
RAM 16 GB

The main idea of this work is built optimizate trajectories and make automatic classification task. All information will be complementary with my personal github account. If you have some specify question write to: jose.avalos@utec.edu.pe

Official information could be find about the sawyer: http://sdk.rethinkrobotics.com/intera/Main_Page

Regards!
